# Top 10 News Reader

## Introduction
This is a top 10 news reader app. It fetches the news from API and reads them.

## Project Setup
Follow the steps below to set up the project:

1. Open Command Prompt (Windows) or Terminal (Linux/macOS).
2. Navigate to the directory where you want to clone the project.
3. Execute the following commands one by one.

### Windows
```
git clone https://github.com/problem-solver-man/top-10-news-reader.git
cd top-10-news-reader
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python news.py
```

## Get API Key
Get the news API key from:
```
https://newsapi.org/
```
